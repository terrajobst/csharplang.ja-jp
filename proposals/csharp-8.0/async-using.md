## <a name="async-using-declaration"></a>非同期 (宣言を使用した)

8\.0 C#では、*非同期の using*ステートメントのサポートが追加されました。 2つの形式があります。 1つには、declaratation を使用するのスコープであるブロック本体があります。 もう1つはローカルを宣言し、ブロックの末尾に暗黙的にスコープが設定されます。 これは仕様のプレースホルダーです。
